# PRODUCT REQUIREMENTS DOCUMENT

## Product Name
Smart LMS

## Vision
Platform pembelajaran modern untuk sekolah dan kampus.

## Goals
- Mempermudah distribusi materi.
- Mempermudah pengumpulan tugas.
- Mempermudah penilaian guru.
- Memantau progres belajar siswa.

## User Roles
### Admin
Mengelola user, kelas, dan mata pelajaran.

### Guru
Mengelola materi, tugas, nilai, feedback.

### Siswa
Belajar, mengumpulkan tugas, melihat nilai.

## Features
- Dashboard
- Materi Pembelajaran
- Assignment
- Submission
- Feedback
- Revision
- Portfolio
- Notification
- Learning Progress
