# PROJECT RULES

## Core Principles
- LMS umum untuk semua mata pelajaran.
- Role: Admin, Guru, Siswa.
- Semua tugas dinilai oleh Guru.
- Tidak ada auto grading sebagai penilaian akhir.
- Coding assignment hanya media pengerjaan dan pengumpulan.
- Sistem harus mendukung materi dan tugas non-coding.

## Supported Assignment Types
- Essay
- File Submission
- Coding Assignment
- Project Assignment
- Quiz

## Main Modules
- Authentication
- Class Management
- Material Management
- Assignment Management
- Submission Management
- Assessment & Feedback
- Portfolio
- Notification
- Learning Progress

## Development Rules
- Responsive UI
- Role-based access control
- Audit trail untuk submission dan revisi
- Modular architecture
